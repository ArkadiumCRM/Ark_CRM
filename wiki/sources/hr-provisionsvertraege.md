# HR · Provisionsverträge "Praemium Victoria"

Jährlich neu abgeschlossene Zielvereinbarung je MA, integraler Bestandteil des Arbeitsvertrags. 2 Varianten im Set (Candidate Manager vs Team Leader); der Researcher-Vertrag kann Praemium Victoria ebenfalls referenzieren.

**Vorlagen-Jahr:** April 2024 – Dezember 2024 (9-Monats-Zeitraum als Beispiel). Bei Jahresablauf wird neue Provisionsvereinbarung geschlossen.

## Gehaltsstruktur (Beispiel Candidate Manager 2024 Q2–Q4)

| Position | Betrag |
|----------|--------|
| Fixlohn (9 Monate) | CHF 49'800 |
| Variabel bei 100% ZEG | CHF 30'400 |
| Spesenpauschale | CHF 300/Mt = CHF 2'700 (9 Mt) oder 3'600/J |
| **Total @ 100% ZEG** | **CHF 83'800** (inkl. Karenzentschädigung) |

**Budget-Ziel Candidate Manager (Architecture):** CHF 320'000 realisierte Net Fees (9 Mt).
**Budget-Ziel Team Leader REM (Real Estate Management):** CHF 850'000 realisierte Net Fees (inkl. unterstellte MA).

## §4 Funktionen + Provisionsberechtigung

Ein MA kann bei einem Vertragsabschluss gleichzeitig **nur eine** der folgenden Rollen einnehmen:
- Candidate Manager (**Kandidatenhalbpunkt**)
- Account Manager (**Kundenhalbpunkt**)
- Team Leader (**Teambudget**, inkl. unterstellte MA)

**Ausschluss:** Nicht zusätzlich als Researcher beim selben Abschluss.

## §5 Vertragsabschluss-Definitionen

### §5.1 Beratungsgeschäft (erfolgreich, wenn **alle** erfüllt):

1. Schriftlicher Auftrag Arkadium ↔ Kunde (Beratungen, Wissensvermittlung, Schulungen)
2. Arkadium hat Leistung vollständig erbracht
3. Rechnung durch Account Manager gestellt
4. Kundenunternehmen hat Honorar vollständig beglichen
5. **Realisierter Umsatz**

### §5.2 Vermittlungsgeschäft (erfolgreich, wenn **alle** erfüllt):

1. Gültiger schriftlicher Arbeitsvertrag Kandidat ↔ Kunde
2. Rechnung durch Account Manager gestellt
3. Kandidat hat Stelle angetreten **und Probezeit bestanden**
4. Kundenunternehmen hat Vermittlungshonorar vollständig bezahlt
5. **Garantie-/Rückzahlungsfrist abgelaufen** oder schriftlich ausgeschlossen
6. **Realisierter Umsatz**

### Garantie-Rückzahlung

Bei teilweiser/voller Garantieleistung durch Arkadium: zurückbezahlter Anteil = **nicht realisierter Bruttoumsatz** → keine Provisionsberechtigung · bei bereits erhaltener Provision: Rückzahlungspflicht oder Verrechnung mit künftigen Ansprüchen.

### §5.3 Rollenwechsel-Regel Researcher → Consultant

- **3 Monate** nach Übertritt: Provisionsanspruch auf Vermittlungen, bei denen MA als Researcher Erstansprache + CV erhalten hat, **entfällt**
- CV als Researcher erhalten + innerhalb 3 Mt als Consultant vermittelt → nur Consultant-Anspruch (nicht doppelt)

## §6 Net Fees + Halbpunkt-System

**Net Fees** = Bruttoumsatz − Rabatte − direkt verrechenbare Spesen − Referral Fees − Eintreibungs-/Bestreitungs-Kosten − Extrakosten (z.B. Marketing).

### Vermittlungsgeschäft

- Net Fees **50/50 geteilt**: Candidate Manager (Kandidatenhalbpunkt) + Account Manager (Kundenhalbpunkt)
- Teamleiter erhält Hälfte der realisierten Net Fees ins Teambudget bei CM-Tätigkeit seines Unterstellten (oder eigener) · ganze Net Fees wenn beide Halbpunkte im Team

### Beratungsgeschäft

- Aufteilung nach Funktion + Gewichtung der Tätigkeit **nach GL-Ermessen**

### Konflikt-Fall (gleiche Hierarchiestufe)

- GL entscheidet nach Ermessen über Halbpunkt-Zuordnung · ggf. Teilung · max 100% eines Halbpunktes verteilt · kein Rechtsanspruch aus Kulanz

### §6.1 Honorarstreitigkeiten

- Nachträglich bezahltes Honorar durch Drittperson-Prozess → **ausserordentlicher Ertrag**, nicht im Budget-Ziel
- MA-Anspruch: **5% des Bruttoumsatzes − CHF 500 Managementkosten**

### §6.2 Austritts-Vergütung laufender Prozesse

**Beratungsgeschäft** (laufend = Auftrag geschlossen, Honorar noch nicht bezahlt):
- Nach schriftlich geschlossenem Auftrag: **10–25%** (GL-Ermessen)

**Vermittlungsgeschäft** (laufend = Dossier vorgelegt / Interview geführt / Offer erhalten):
- Vor/nach 1. Kunden-Interview (pers./Tel.): **10–25%**
- Vor/nach 2. Kunden-Interview: **25–50%**
- Konkretes Jobangebot erhalten: **50–75%**
- GL-Ermessen für genauen Satz

→ Net Fees können **nach Austrittszeitpunkt** realisiert werden.

## §7 ZEG-Staffel (Anhang · Provisionsstaffel)

ZEG = Gesamtzielerreichungsgrad = realisierte Net Fees / Budget-Ziel.

| ZEG-Bereich | Provisionssatz |
|-------------|----------------|
| 0–49% | **0%** |
| 50% | 10% |
| 51–59% | +2% pro 1% ZEG (12% / 14% / ... / 28%) |
| 60% | 30% |
| 61–69% | +1% pro 1% ZEG (31% / ... / 39%) |
| 70% | 40% |
| 71–79% | +2% pro 1% ZEG (42% / ... / 58%) |
| 80% | 60% |
| 81–89% | +2% pro 1% ZEG (62% / ... / 78%) |
| 90% | 80% |
| 91–99% | +2% pro 1% ZEG (82% / ... / 98%) |
| **100%** | **100%** |
| 101–149% | +2% pro 1% ZEG (102% / ... / 158%)* |
| 150% | 160% |
| 150%+ | +1% pro 1% ZEG über 150% |

*Hinweis: ab 101% scheinen nur noch +1% pro ZEG, Tabelle zeigt aber 101=102 / 102=104 / 103=106 — also **+2% bei 50–99% und 101–149% in kleinen Schritten**. Kontrolle bei Schema-Implementierung.

**Staffel stimmt mit [[project_commission_model|Commission-Engine Option D]] überein** (gleiche ZEG-Staffel in neuer Engine v1.0).

## §6–§7 Auszahlung

### Quartalsweise Vorschüsse

| Periode | Berechnung | Auszahlung |
|---------|------------|------------|
| Q1 (Jan–Mär) | Soll vs Ist | Ende April |
| Q2 (Apr–Jun) | | Ende Juli |
| Q3 (Jul–Sep) | | Ende Oktober |
| Q4 (Okt–Dez) | Schlussabrechnung | **Ende Februar Folgejahr** (nach revidiertem Jahresabschluss) |

**80/20-Split:** Vorschüsse nur **80% ausbezahlt**, **20% Rücklage** → mit Jahresendzahlung nach allen Zahlen + Garantiefrist.

**Arkadium-Vorbehalt:** kann gesamte Provisionen erst nach Ablauf Garantiefrist auszahlen (Risikomanagement).

### §7.1 Zu-viel-Auszahlung bei Austritt

Zu-viel-bezahlte Provisionen/Vorschüsse können in Quartalen des Folgejahres verrechnet oder vom MA zurückverlangt werden.

## §10 Widerspruchs-Regel

Provisionsvertrag geht anderslautenden Abmachungen vor · sämtliche Änderungen bedürfen Schriftform.
