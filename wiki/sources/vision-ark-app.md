# Vision Ark_App — Heritage-App

Brainstorm-Dokument zu einer **separaten Vision-App** (nicht ARK CRM). Kernidee: Heritage-App als intergenerationale Messplattform — Glaubenssätze, Erfahrungen, Potenziale weiterreichen, damit Kinder nicht bei null anfangen müssen.

## Grundmetapher

- **DNA** = Hardware (Krankheiten, genetische Prägung)
- **Epigenetik** = Betriebssystem (Erfahrungen, Umfeld, Lebensstil)
- **Glaubenssätze** = Software (messbar, veränderbar)

**Intergenerationale Prägung** bisher nicht messbar — Lücke, die App schliessen soll.

## Messbare Dimensionen (Potenzial-Entfaltung)

### Ressourcen / Potenzial
- Diagnostik, Analysen
- Ressourcen, die entfaltet werden können

### Charakter-Basis (bis 15 Jahre)
- Trauma-Screening (erste 15 Jahre → heutzutage gut messbar → Software-Layer)
- Erfahrungen / Umfeld → Epigenetik
- Schulbildung / Wissen / Allgemeinbildung / Fachwissen → beeinflusst Epigenetik + Glaubenssätze + Charakter
- Bedürfnisse / Leidenschaft / Hobbies / Freude → Charakter-Teil

### Ab 15 Jahren (Wissen + Charakter-Messung)
- Verhaltensmuster, Motive, **Big 5**, Werte
- Kognitive Leistungen
- **9 Intelligenzen**
- Werte / Motive / **Ikigai**

### Berufs-Koppelung
- **Eignungstests** erweiterbar
- **Hackman-Oldham Job-Charakter-Modell** → Matching Job ↔ Charakter
- **Selbstwirksamkeit** (Bandura) → Glaube an eigene Handlungsfähigkeit
- **Lebenslinienanalyse** → Ausrichtung / Orientierung / Sinnhaftigkeit
- **Holland-Theorie** (Berufswahl)

### Erweiterungs-Theorien
- Fach-/Soft-/Hardskills
- **Motivationsparameter-Score (MPS)**
- Maslow · Rubikon · Herzberg (Motivation + Hygienefaktoren) · RPM (Tony Robbins)
- **Coping-Strategien**
- **Safety**: team/sozial (Edmondson-Modell, Amy-Theorie)
- Neurowissenschaft / **Neuroscan**

## Kernbild: Trichter der Optionen

Ziel: Volatilität der Lebensamplitude reduzieren. Statt von Rand zu Rand (Option-Trichter) pendeln → Trichter verengen. Weniger Zeitverlust, mit 80 weiter fortgeschritten.

## Abgrenzung zum ARK CRM

- **Ark CRM** = operatives Executive-Search-System
- **Vision Ark_App (Heritage-App)** = eigenständiges Produkt, Potenzial-/Lebens-Diagnostik-Plattform

Überlappungen denkbar mit [[assessment]] (DISC, EQ, Motivatoren), [[diagnostik-assessment]], [[matching]] — aber fundamental anderer Scope (Lebenslauf-Begleitung vs. Recruiting-Transaktion).

## Offene Fragen

- Wie Messbarkeit von Glaubenssätzen/Intergenerationaler Prägung operationalisieren?
- Separate Infrastruktur oder CRM-Anbau?
- Zielgruppe: Privatpersonen / Familien / Coaches / HR?
- Datenschutz/DSGVO bei intergenerationaler Datenhaltung?
- MVP-Scope: welche der 15+ Theorien zuerst?
